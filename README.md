# Laptop Price Prediction

This project focuses on predicting laptop prices using machine learning regression techniques.

## Dataset
- Source: Kaggle – Laptop Price Prediction
- Features:
  - Brand
  - Processor Speed
  - RAM Size
  - Storage Capacity
  - Screen Size
  - Weight

## Methodology
- Exploratory Data Analysis (EDA)
- Data preprocessing and encoding
- Train-test split
- Linear Regression
- Random Forest Regression

## Results
Linear Regression achieved the best performance due to strong linear relationships in the dataset.
The high R² scores indicate that the dataset may be synthetic or rule-based.

## Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## How to Run
1. Clone the repository
2. Install required libraries
3. Run the Jupyter Notebook
